# Review Command
Бизнес логик болон API endpoint-уудыг дараах шалгуураар хянана уу:
1. **Security**: SQL Injection-оос хамгаалагдсан уу? (Prepared statements ашигласан уу?)
2. **Robustness**: Input validation хийгдсэн үү? (Title хоосон биш үү?)
3. **Error Handling**: Try/catch ашиглаж, хэрэглэгчид ойлгомжтой алдааны мессеж буцааж байна уу?
4. **Clean Code**: Service болон Route давхаргууд зөв тусгаарлагдсан уу?
