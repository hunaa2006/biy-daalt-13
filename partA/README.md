# Part A: Planning 

Энэхүү хавтаст "Personal Task Tracker" төслийн бэлтгэл шатны бүх баримт бичгүүд байрлаж байна. Бид төслийг эхлүүлэхээс өмнө архитектур, технологийн сонголт болон AI-тай ажиллах дүрмүүдээ тодорхойлсон.

## Төлөвлөлтийн файлууд

1. **[PROJECT.md](./PROJECT.md)** - Төслийн зорилго, хамрах хүрээ (Scope).
2. **[STACK-COMPARISON.md](./STACK-COMPARISON.md)** - Технологийн сонголтын харьцуулалт.
3. **[ARCHITECTURE.md](./ARCHITECTURE.md)** - Системийн бүтэц (Mermaid диаграмм).
4. **[adr/0001-stack.md](./adr/0001-stack.md)** - Архитектурын шийдвэрийн тэмдэглэл (ADR).
5. **[ai-sessions/plan.md](./ai-sessions/plan.md)** - Төлөвлөлтийн үед AI-тай харилцсан түүх.

## Төлөвлөлтийн үр дүн
- **Backend:** Node.js (Express)
- **Database:** SQLite
- **Frontend:** HTML/JS (Vanilla)
