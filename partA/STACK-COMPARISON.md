# Stack харяцуулалт - Personal Task Tracker

## Харьцуулсан stack-ууд
| Stack | Технологи |
|-------|-----------|
| A | Node.js + Express + SQLite |
| B | Python + FastAPI + PostgreSQL |
| C | Node.js + Express + MongoDB |

## Харьцуулалт

### Stack A: Node.js + Express + SQLite
* **Давуу тал:** Setup хамгийн хурдан. SQLite нь файл хэлбэрээр хадгалагддаг тул нэмэлт DB сервер суулгах шаардлагагүй. Нэг хэл (JavaScript) ашиглах тул хурдан хөгжүүлнэ.
* **Сул тал:** Проект томорвол өгөгдлийн сангийн гүйцэтгэл хязгаарлагдмал.

### Stack B: Python + FastAPI + PostgreSQL
* **Давуу тал:** OpenAPI (Swagger) автоматаар үүсдэг. Python-ий бичиглэл цэвэрхэн.
* **Сул тал:** PostgreSQL тохируулах нь бие даалтын хугацаанд нэмэлт ачаалал болно.

### Stack C: Node.js + Express + MongoDB
* **Давуу тал:** Schema-less буюу уян хатан өгөгдлийн бүтэцтэй.

## Сонголт: Stack A — Node.js + Express + SQLite

### Шалтгаан
1. Setup хурдан, нэг хэлтэй (JS)
2. SQLite - тохиргоо хэрэггүй, репод шууд
3. Jest-ээр >=10 test хялбар
4. OpenAPI spec гараар бичих шаардлагатай тул FastAPI-ийн давуу тал хэрэггүй
