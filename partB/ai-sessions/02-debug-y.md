# AI Session Log: 02 - Алдаа засах ба Шүүлтүүр нэмэх

**Огноо:** 2024-05-09
**Зорилго:** Даалгавруудыг статусаар (pending/completed) болон priority-оор шүүхэд гарсан алдааг засах.

### 1. AI-д өгсөн Prompt
> "GET /tasks endpoint дээр статус болон priority-оор шүүх функц ажиллахгүй байна. Query параметрүүдийг хэрхэн зөв авч, SQL query-д дамжуулах вэ?"

### 2. AI-ийн хариу ба тусламж
AI надад динамик SQL query үүсгэх аргыг зааж өгсөн. Хэрэв хэрэглэгч шүүлтүүр сонгоогүй бол бүх өгөгдлийг, сонгосон бол тухайн утгаар нь шүүх `WHERE` нөхцөлийг нэмж өгсөн.

### 3. Гарсан асуудал ба шийдэл (Hallucination)
AI эхлээд файлын нэрийг `taskServices.js` (олон тоон дээр) гэж дуудаж алдаа гаргасан. Миний файл `taskService.js` (ганц тоо) байсан тул `require` хийхэд алдаа зааж байсныг би өөрөө олж зассан. Энэ нь AI-ийн нэршил дээр гаргадаг жижиг алдаа (hallucination) байлаа.

### 4. Үр дүн
- Шүүлтүүр (Filtering) болон Хайлт (Search) амжилттай ажилладаг болсон.
- Алдаа засах явцад кодын бүтэц илүү ойлгомжтой болсон.
