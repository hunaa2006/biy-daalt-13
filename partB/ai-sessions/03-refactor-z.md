# AI Session Log: 03 - Тест бичих болон Код цэгцлэх

**Огноо:** 2024-05-10
**Зорилго:** Jest болон Supertest ашиглан 10-аас доошгүй unit тест бичих.

### 1. AI-д өгсөн Prompt
> "Миний TaskService-д зориулж Jest unit тестүүд бичиж өг. Нийт 10 тест байх ёстой бөгөөд 'Happy path' болон 'Edge cases' (хоосон утга, буруу ID гм)-ийг шалгах хэрэгтэй."

### 2. AI-ийн хариу ба тусламж
AI надад `tests/taskService.test.js` файлыг үүсгэж өгсөн. Тест бүрийн өмнө өгөгдлийн санг цэвэрлэх `beforeEach` функцийг ашиглахыг зөвлөсөн нь маш хэрэгтэй байлаа.

### 3. Гарсан асуудал ба шийдэл
Тестүүдийг ажиллуулахад `tasks.db` файл ашиглагдаж байгаа тул өгөгдөл давхцах асуудал гарсан. AI-ийн зөвлөсний дагуу тестэнд зориулж "In-memory database" эсвэл тест бүрийн дараа өгөгдлийг устгах SQL ашиглаж шийдсэн.

### 4. Үр дүн
