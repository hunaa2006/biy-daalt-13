# AI Session Log: 01 - Үндсэн функцүүд болон Өгөгдлийн сан

**Огноо:** 2024-05-08
**Зорилго:** Төслийн backend суурийг тавьж, Task үүсгэх, унших, засах, устгах (CRUD) үйлдлүүдийг хийх.

### 1. AI-д өгсөн Prompt
> "Node.js (Express) болон SQLite (better-sqlite3) ашиглан Task Tracker-ийн CRUD үйлдлүүдийг хийж өг. Өгөгдлийн сангийн бүтэц нь id, title, description, status, priority, due_date-тэй байна. Код нь Route болон Service давхаргад хуваагдсан байх ёстой."

### 2. AI-ийн хариу ба тусламж
AI надад `database.js` холболтын файл болон `taskService.js` доторх SQL query-нүүдийг бичиж өгсөн. Ингэхдээ:
- **better-sqlite3** ашиглан синхрон байдлаар өгөгдлийн сантай харилцахыг зөвлөсөн.
- **Prepared statements** ашиглан SQL Injection-оос хамгаалсан.

### 3. Гарсан асуудал ба шийдэл
Анх AI бүх кодыг нэг файл дотор (app.js) өгсөн боловч би бие даалтын архитектурын шаардлагын дагуу `src/routes/` болон `src/services/` болгон салгахыг хүссэн. Үүний үр дүнд код илүү цэгцтэй болсон.

### 4. Үр дүн
- `taskService.js` болон `tasks.js` файлууд бэлэн болсон.
- Өгөгдлийн сан автоматаар үүсдэг болсон.
